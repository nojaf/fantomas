ILNativeType Type

Namespace: Fantomas.FCS.AbstractIL
Assembly: Fantomas.FCS.dll
Parent Module: IL
Base Type: obj
All Interfaces: IStructuralEquatable , IComparable<ILNativeType> , IComparable , IStructuralComparable

Native Types, for marshalling to the native C interface. These are taken directly from the ILASM syntax. Most of these are listed in the CLI ECMA-335 Spec (Partition II, 7.4).
